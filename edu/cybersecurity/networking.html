protocols

TCP

TCP (Transmission Control Protocol) is when two computers make an established connection making sure that no data is lost. TCP prefers accuracy at the loss of speed.

UDP

UDP (User Datagram Protocol) is when data is thrown at the receiving computer, possibly losing data at some times. UDP prefers speed at the loss of accuracy.

OSI Model

OSI or Open Systems Interconnection outlines the 7 main layers to networking.

7 Application

  • User View
  • UI & UX
  • “providing an interface for them to use in order to transmit data”

6 Presentation

  • Data is passed down from the application layer (7).
  • Data is usable for local applications but not it isn't always the standard, so data is standardised for the reciving computer.
  • Data is encrypted, compressed and any other transformations necssary.

5 Session

  • Data is passed down from the presentation layer (6).
  • The session layer looks to see if it can set up a connection with the other computer across the network.
  • The session layer sets up, manages and terimates connections.
  • The session layer is also respobsible for managing multiple connections at the same time, allowing diffreint network applications to run at the same time.

4 Transport

  • Data is passed down from the session layer (5).
  • The transport layer selects what protocol to use.
  • The transport layer divides the transmission into bite-sized pieces.

3 Network

  • Data is passed down from the transport layer (4).
  • The network layer uses IP addresses to locate the destination of your request.
  • Logical Addressing

2 Data Link

  • Data is passed down from the network layer (3).
  • The data link layer focuses on the physical addressing of the transmission.
  • The data link layer adds the MAC address of the device to the packet (data).
  • The data link layer is also resposible for error detection (& corruption) and presenting the data in a format suitable for transmission.

1 Physical

  • Data is passed down from the data link layer (2).
  • The physical layer is right down to the hardware of the computer.
  • It's the job of the physical layer to convert the binary data of the transmission into signals and transmit them across the network.
