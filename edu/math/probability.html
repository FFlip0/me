example questions

Q1

P(A) = 0.7,    P(B) = 0.5,    P(A ∩ B) = 0.4
Find P(A ∪ B)

answer

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
P(A ∪ B) = 0.7 + 0.5 − 0.4
P(A ∪ B) = 0.8

Q2

P(A) = 0.4,    P(B) = 0.5,    P(A ∩ B) = 0.85

i) Find P(A ∩ B)

answer

P(A ∩ B) = P(A) + P(B) − P(A ∪ B)
P(A ∩ B) = 0.4 + 0.8 − 0.85
P(A ∩ B) = 0.35

ii) Find P(A` ∩ B`)

answer

P(A′ ∩ B′) = 1 − P(A ∪ B)
P(A′ ∩ B′) = 1 − 0.85
P(A′ ∩ B′) = 0.15
